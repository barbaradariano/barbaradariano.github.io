transition-property

Nome da propriedade que vai sofrer a transição


              .example1 { transition-property: background-color; }

Coloca todas as prpriedades do CSS como transicionáveis


              .example2 { transition-property: all; }

Translate

transform: translate(ax[, ay]);

Animações

Anima a transição de um estilo de configuração para outro

Animação

Dois componentes

  1. descrição da animação no CSS
  2. conjunto de "keyframes" que indica o inicio e o fim da animação

Animações

animatable 
@-webkit-keyframes background-color{
  from { background-color: slategray }
  to { background-color: black }
}
#background-color {
  -webkit-animation: background-color 1s infinite alternate;
  background-color: slategray
}
